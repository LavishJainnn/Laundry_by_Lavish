--> Laundry Hero – Online Laundry Service Website

Laundry Hero is a modern, responsive web application designed for booking laundry services online. It allows users to select services, add them to a cart, calculate total cost, and place an order using EmailJS for email confirmation. The UI is clean, user-friendly, and optimized for all screen sizes.
Live Demo: https://laundry-by-lavish.netlify.app/


--> Features

- Smooth scroll navigation
- Hero section with Call-To-Action button
- Dynamic services listing
- Add / Remove items from cart
- Live cart total calculation
- Order booking form with validation
- Email confirmation using EmailJS
- Fully responsive design (Desktop, Tablet & Mobile)
- Clean UI with modern layout
- Social media integration


--> Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Responsiveness  
- **JavaScript (Vanilla JS)** – Logic & Cart System  
- **EmailJS** – Sending booking details to email  
- **Ionicons** – Social icons

--> How It Works

1. User selects laundry services.
2. Services are added to a cart dynamically.
3. Total price is calculated automatically.
4. User fills in booking details.
5. Order is sent to email via EmailJS.
6. Confirmation message is shown on successful booking.


--> Setup & Usage

1. Clone the repository: git clone https://github.com/your-username/your-repo-name.git
2. Open the project folder: cd laundry
3. Open index.html in your browser.
4. Setup EmailJS:
    Create an account on https://www.emailjs.com
    Get:
        Service ID
        Template ID
        Public Key

Replace them in the script.
